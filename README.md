# libmedusa

A fork of medusalog program as a library for independent projects

## How to build?

~~~bash
libmedusa> mkdir build
libmedusa> cd build
build> cmake -DCMAKE_BUILD_TYPE=Release ..
~~~

## How to run the test program?

~~~
build> ./test/medusatest
~~~
